Executive Summary
경영진 요약
"상품 짤 때마다 계산하기 너무 귀찮고 힘들고 금액이 전혀 안 맞아서" — 워터트리 내부의 원체험에서 시작. 우리도 필요한 기능이고, 같은 통증을 다른 인바운드 여행사들도 매일 겪고 있다. 워터트리가 자체 제작 · 무료 공개로 이 통증을 걷어낸다.

왜 지금인가

  • 엑셀 수식의 한계 (현장 직접 관찰) — 여행사들이 자체 제작한 IF·VLOOKUP 수식 뒤엉킨 시트를 매일 쓰고 있음. 눈으로 보기에도 엄청 복잡. 실수 1번이 1인당 수만원 오차. 대체 시장이 이미 입증됨.
  • AI·노코드 붐 — 기획자가 코드 없이 바로 가격표를 뽑고 공유하는 경험이 표준 기대치가 됨.
  • 인바운드 V자 회복 — 2025년 이후 동남아·일본·중국 관광객 재확대, 패키지 수요 폭증.
  • B2B 도매 경쟁 심화 — 공급가·판매가를 즉석에서 보여줄 수 있는 파트너가 거래에서 유리.
  • 경쟁 공백 — 한국 인바운드 특화 단가 SaaS는 현재 시장에 없음. 선점자가 표준이 됨.

Vision & Background
비전 & 배경

문제 정의 — 원체험 + 현장 관찰

현장의 목소리 ① — 워터트리 내부 (대표)
"상품 짤 때마다 계산하기 너무 귀찮고 힘들고, 금액이 전혀 안 맞아서 매번 다시 해야 한다.
이걸 다른 여행사들도 똑같이 느끼고 있을 것이다."
현장의 목소리 ② — 다른 여행사 관찰 (Field Evidence)
"어떤 여행사는 이런 화면이 없어서 엑셀로 수식을 만들어서 계산하고 사용하고 있던데, 엄청 복잡해 보였다."
→ 시장 가설이 아닌 실제 관찰 증거(Market Validation). 현장에서 IF·VLOOKUP·SUMPRODUCT가 수식 수십 개로 얽힌 시트를 매일 쓰고 있다. 이 복잡성을 한 화면으로 대체하는 것이 우리 제품의 존재 이유.

워터트리 상품기획팀은 외국인 대상 패키지를 구성할 때마다 엑셀 시트에 의존한다. 차량·가이드·관광지 단가를 손으로 조합해 원가를 산출하며, 인원수가 10명 → 20명으로 바뀌면 공통비 분담 로직을 전부 재계산해야 한다. 언어별 가이드 단가(영/일/중/대만/러/몽)가 수시로 변동하고, 관광지 추가·삭제마다 행을 밀고 당긴다. 기획 속도가 떨어지고, 실수 하나가 1인당 수만원 오차로 이어진다. 금액이 안 맞아 재작성하는 과정만 하루 수 시간.

이것은 워터트리만의 문제가 아니다. 한국 인바운드 여행사 2,000여 곳, 오프라인 여행사 수천 곳, 프리랜서 가이드 수만 명이 같은 도구로 같은 고통을 반복하고 있다. 이 시장에 맞는 SaaS는 없다. 있는 것은 일반 회계 소프트웨어, 엑셀, 클룩·트립어드바이저 같은 고객 대상 OTA뿐.

★ 자체 제작 + 무료 공개 — 이중 동기
① 우리도 필요한 기능: 워터트리 내부 기획팀이 매일 쓸 도구. 외주 없이 자체 제작.
② 같은 문제는 업계 공통: 우리가 해결하면 다른 여행사들도 쓸 것. 그래서 무료 공개. 서비스를 팔아 돈을 벌지 않고, 업계 표준 도구로 만들어 별도 수익(광고·데이터 리포트)을 얻는다.
③ 오프라인 영업 안 함: 기존 B2B 네트워크 동원 없이 100% 검색 유입으로만 성장. 마케팅비 0원.

최종 비전

인바운드 여행사·오프라인 여행사·상품기획자가 로그인 → 코스 설계 → 단가표 → 공유를 10분 안에 끝내는 PWA. 모바일·PC·태블릿 어디서나 실시간 동기화, 오프라인에서도 동작, 카톡·이메일로 즉시 공유. 워터트리는 자체 기획 속도를 올리고, 공개 사용자는 무료로 도구를 쓰며, 검색 자연 유입으로 수만 사용자가 쌓이면 광고·상품 유통·데이터 리포트로 수익을 창출한다.

운영 철학

  • 80% 원칙 — 기술 완벽보다 빠른 시장 출시. Phase 0은 2~3주로 내부 사용 검증.
  • 연 고정비 ~2만원 — 도메인만 구매. Supabase·Vercel·카카오·Sentry 전부 무료 티어.
  • 무료 공개 — 서비스 이용은 평생 무료. 수익은 광고·리포트로만.
  • 검색 유입 Only — 오프라인 영업·홍보·B2B 네트워크 동원 없음. SEO와 서비스 자체 가치로만 성장.
  • AI로 개발비 0원 — Claude 기반 AI 개발. 외주 없음.

★ 우리가 파고들 공백 — 3-way Gap
① 가격 공백: 영세 여행사는 월 $99+예약 수수료 못 냄. 우리는 평생 무료.

DMC·인바운드 공급망 특수성 (Arival·DMC Quote 자료)

  • Net Rate(공급가) vs Rack Rate(소매): 공급가는 공개가보다 15~40% 낮은 B2B 전용 가격. 호텔 예: $150 rack → $100 net.
  • 4단계 마진 분배: Hotel → DMC(7~10%) → Outbound 도매(20~25%) → Retail 에이전시(7~10%). 우리 앱의 netMarginRate · retailMarginRate · 채널 마진이 정확히 이 체인을 모델링.
  • 한국 시장: 한국 인바운드 커미션 9~20% 범위. 정부 2026 정책 "가격 투명성 강화" 추진 — 우리 도구가 여기에 부합.
